>### Setup

- On the front-end.
  > `cd search_engine_FE`
  > npm install
  > npm start
- on the back-end.
  > `cd search_engine_BE`
  > bundle install
  > rails s

### To run tests

- backend - `rspec ./spec/`

### run linters
<!-- For eslint errors -->
- frontend 
  > `npx eslint .`
  
<!-- To correct eslint errors -->
  >`npx eslint . --fix`
  
<!-- For styelint -->
  >`npx stylelint "**/*.{css,scss}"`
  
<!-- To correct stylelint errors -->
  >`npx stylelint "**/*.{css,scss}" --fix`

- backend
  > `rubocop -A`
